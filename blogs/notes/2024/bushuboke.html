超详细，无服务器部署个人博客VuePress + vuepress-reco + Github Pages

wadewei2024/05/29运维VuePress

介绍

VuePress 是一个以 Markdown 为中心的静态网站生成器。你可以使用 Markdown在新窗口打开open in new window 来书写内容（如文档、博客等），然后 VuePress 会帮助你生成一个静态网站来展示它们。

VuePress 诞生的初衷是为了支持 Vue.js 及其子项目的文档需求，但是现在它已经在帮助大量用户构建他们的文档、博客和其他静态网站。

话不多说，直接上

再提一句，首先你要有安装node，我这里的版本是v20.12.2，另外我直接选择vuepress-recoopen in new window主题提供的脚手架去初始化博客项目，话不多说，跟着做就行了

快速开始

  1. 创建文件夹blog

  2. win+r，输入cmd，打开dos窗口

  3. 创建博客项目

    # 安装@vuepress-reco/theme-cli
npm install @vuepress-reco/theme-cli@1.0.7 -g
# 初始化项目 注意选择2.x的版本
theme-cli init
# 安装所有依赖的包
npm install
# 启动项目
npm run dev

  4. 简简单单4步，博客就搭建好了

通过Github Pages发布自己的博客网站

  1. 编译博客项目

    # 编译项目，编译后的文件生成在\blog\.vuepress\dist下
npm run build

  2. 登录githubopen in new window

  3. 创建一个项目，Repository name 为 {username}.github.io

  1. 上传编译后的文件，到dist目录下

    git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/WaderWei/WaderWei.github.io.git
git push -u origin main

  2. 查看生成的Github Pages的urlopen in new window

  3. 这样一个博客网站就部署好了，赶紧开始写博客吧
